DNAcopy包的作用是处理芯片矩阵或二代测序的数据，通过read数来判断染色体上DNA拷贝片段的边界，获得DNA片段拷贝信息。

加载数据

library(DNAcopy)
#加载数据集
data(coriell)
#>         Clone Chromosome Position Coriell.05296 Coriell.13330

#> 6  RP11-51b04          1     7000     -0.006770      0.094144

可以看到数据集包含探针编号(Clone), 所属染色体(Chromosome), 在染色体上的位置(Position), 样本检测数据(Coriell.05296, Coriell.13330)。DNAcopy支持logratio和binary两种数据格式。

生成CNA.object对象

# 可以对单个的样本建立CNA.object对象
CNA.object <- CNA(cbind(coriell$Coriell.05296),
                  coriell$Chromosome,coriell$Position,
                  data.type="logratio",sampleid="c05296")
#> Warning in CNA(cbind(coriell$Coriell.05296), coriell$Chromosome, coriell$Position, : array has repeated maploc positions

# 也可以对多个样本一起建立CNA.object对象
CNA.object2 <- CNA(coriell[,c('Coriell.05296','Coriell.13330')],
                  coriell$Chromosome,coriell$Position,
                  data.type="logratio",sampleid=c("c05296",'c13330'))
#> Warning in CNA(coriell[, c("Coriell.05296", "Coriell.13330")], coriell$Chromosome, : array has repeated maploc positions


# 注意如果染色体中有X/Y,需要预先对数据进行排序
# coriell$Chromosome<- ordered(coriell$Chromosome, levels=c(1:22,"X","Y"))

CNA函数包含以下参数：

参数 说明
genomdat 矩阵，行为探针/基因名称，列为样本
chrom 染色体
maploc 在染色体上的位置
data.type 数据类型, 可选logratio或binary
sampleid 样本名称
presorted 是否预选对染色体进行排序,默认为FALSE

此外可以对数据进行平滑处理，去除异常值

smoothed.CNA.object <- smooth.CNA(CNA.object)

识别变异区间

segment函数应用循环二进制分割(CBS)算法来识别并划分变异区间

segment.smoothed.CNA.object <- segment(smoothed.CNA.object, verbose=1)
#> Analyzing: c05296
参数 说明
x 矩阵，行为探针/基因名称，列为样本
weights 探针的权重
alpha 显著性水平
p.method 显著性检验方法
min.width 最小片段的长度
